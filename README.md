# Magento Amasty Home Hreflang Fix

Production-safe Magento 2 fix for incorrect HTML hreflang URLs generated by Amasty on multilingual stores.

## Goal

This repository contains a custom Magento 2 module that fixes homepage hreflang URLs when Amasty outputs the CMS page identifier URL instead of the real store base URL.

Example:

Wrong:
- `/it/athlete2-home-fi`
- `/en/athlete2-home-fi`

Correct:
- `/it/`
- `/en/`
